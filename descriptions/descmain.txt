Система расчета рейтинга активности участников клана нужна для кика неактивных игроков.

Активность рассчитывается только для текущих участников клана.

Рейтинг активности рассчитывается по периодам.
    Начало периода - 11 число месяца
    Конец периода - 10 число следующего месяца
    Длительность периода - месяц

Когда участник клана может быть исключен:
    В конце периода по результатам подсчета активности за период (После ЛВК)
    Во время периода при наступлении определенных условий()



Критерии активности:
    *Участие в КВ - api
    *Участие в ЛВК - api
    *Участие в ИК - png input
    *Участие в рейдах - api
    *Количество пожертвованных войск - api
    *Количество дней оффлайн - api + большие расчеты
    *Прокачка деревни - api
    *Пожертвования столицы - api

Условия, при которых участник клана может быть исключен до окончания периода:
    Результат ИК больше нуля, но меньше 200.
    Пропуск 3 КВ подряд будучи участником - относительный показатель(прим. 2 пропуска подряд будучи участников, затем перед третьей кв красный знак - показатель сбрасывается до 0)
    Неактив 7 дней подряд при полном клане - относительный показатель
    Пропуск 3 рейдов подряд - относительный показатель

Условия, при которых участник клана может быть исключен по окончанию периода(абсолютные показатели):
    Пропуск всех рейдов
    Результат ИК менее 1000
    Пропуск ЛВК при заявленном участии
    Общее количество дней оффлайна(дни, в которые игрок не заходил в игру) более 10
    Пожертвовано 0 войск
    0 участий в КВ
    Ни одна из деревень не прокачивалась


В КВ по умолчанию участвуют все, у кого нет красного знака в профиле. Исключение: игрок выбрал "Нет" в заявке на КВ в чате, или игрок предупредил в чате.
Состав ЛВК набирается по результатам активности до начала ЛВК

Графы относительных показателей:
    Количество пропущенных КВ подряд, будучи участником -> число. Больше - хуже
    Последний раз онлайн -> число. Больше - хуже
    Количество пропущенных рейдов подряд -> число. Больше - хуже

Графы, по которым отслеживается активность отдельного игрока(абсолютные показатели):
    Результат ИК -> число. Больше - лучше
    Количество рейдов, в которых участвовал -> число. Больше - лучше
    Отношение кол-ва атак в рейдах к кол-ву доступных атак -> число(0<=x<=1). Больше - лучше
    Количество пожертвованных войск -> число. Больше - лучше
    Количество элементов, уровень которых повысился -> число. Больше - лучше
    Количество дней оффлайн(Дни, в которые не изменился ни один из показателей активности) -> число. Меньше - лучше
    Количество пожертвований столицы -> число. Больше - лучше
    Участие в ЛВК -> булево. True - лучше
    Количество атак в ЛВК -> число. Больше - лучше
    Количество участий в КВ, в которых не проводил атаки -> число. Меньше - лучше
    Количество участий в КВ, в которых провел хотя бы 1 атаку -> число. Больше - лучше
    Отношение кол-ва участий в КВ(с атаками) к общему числу КВ -> число(0<=x<=1). Больше - лучше




Версии приложения:
    *1 - отслеживается активность только в текущем периоде
    2 - история активности по периодам
    3 - отслеживание эффективности
