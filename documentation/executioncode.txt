execution code - четырёхзначные положительные числа в формате ABXX, где:
A (первая цифра) — основной компонент системы.
B (вторая цифра) — подсистема внутри компонента (например, интерфейс, логика, работа с данными).
XX (две последние цифры) — конкретный код состояния в данной подсистеме.

Распределение диапазонов по компонентам
1xxx	Десктопное приложение	Всё, что относится к настольной версии
2xxx	Модуль данных	DataHandler, DataGetter, Parser, Saver, Loader, UpdateDecider
3xxx	Сервер	Будущий бэкенд
4xxx	Мобильное приложение
5xxx	Сайт (веб-приложение)
0xxx, 9xxx	Зарезервировано	Для внешних API или общих ошибок


Диапазон интерфейса десктопного приложения (19xx)

1900-1909	Общие ошибки UI	Инициализация, неизвестные ошибки
    1900 – Успех (операция в UI выполнена)
    1901 – Неизвестная ошибка интерфейса
    1902 – Ошибка инициализации виджета
1910-1919	MembersTable	Таблица участников
    1911 – Не удалось загрузить данные для таблицы (пустой список)
    1912 – Ошибка при заполнении строк таблицы
    1913 – Неверный формат данных для отображения в таблице
1920-1929	PlayerFrame	Карточка игрока
    1921 – Не найден выбранный игрок в данных карточки
    1922 – Ошибка при отображении активности (некорректное значение)
    1923 – Не удалось обновить информацию об игроке
1930-1939	ExtraPanel	Панель дополнительной информации
    1931 – Нет данных для отображения дополнительной информации
    1932 – Ошибка при заполнении таблицы ЛВК (cwl)
    1933 – Ошибка при заполнении таблицы КВ (cw)
    1934 – Ошибка при заполнении таблицы рейдов (raids)
    1935 – Неверный формат данных для extra‑панели
1940-1949	StartupPanel	Стартовая панель (ввод тега)
    1941 – Введён некорректный тег клана (не проходит валидацию)
    1942 – Ошибка при отправке сигнала с тегом
    1943 – Ошибка при сбросе панели после успешного ввода
1950-1959	Взаимодействие	Сигналы, нажатия, выбор элементов
    1951 – Ошибка при выборе элемента в таблице (itemSelectionChanged)
    1952 – Ошибка при нажатии кнопки «Подробнее»
    1953 – Не удалось обработать сигнал от кнопки
1960-1969	Отображение данных	Форматирование, цвета, заполнение
    1961 – Ошибка форматирования текста (например, даты)
    1962 – Ошибка применения цветовой индикации
    1963 – Проблема с обновлением содержимого виджета
1990-1999	Прочие ошибки	Резерв
    1990 – Ошибка, не подпадающая под другие категории

Диапазон модуля данных (2xxx)

20xx	Общие коды модуля	Успех, частичный успех, неизвестная ошибка
    2000 – Успех (операция выполнена полностью)
    2001 – Частичный успех (например, часть данных не обработана)
    2002 – Операция не требуется (данные актуальны)
    2003 – Неизвестная внутренняя ошибка
21xx	DataHandler	Внешние коды, возвращаемые в UI
    2100 – Успех (данные получены/обновлены/сохранены)
    2101 – Обновление не требуется (данные актуальны)
    2102 – Частичный успех
    2103 – Ошибка получения данных от DataGetter
    2104 – Ошибка парсинга (DataParser)
    2105 – Ошибка сохранения (Saver)
    2106 – Ошибка загрузки (Loader)
    2107 – Ошибка проверки свежести (UpdateDecider)
    2108 – Неизвестная внутренняя ошибка
22xx	DataGetter	Получение данных из внешнего источника
    2200 – Данные успешно получены
    2201 – Ошибка сети / таймаут
    2202 – Тег клана не найден
    2203 – Доступ запрещён (API‑ключ и т.п.)
    2204 – Ошибка сервера (5xx)
    2205 – Пустой ответ
23xx	DataParser	Парсинг и валидация
    2300 – Данные успешно распарсены
    2301 – Неверный формат (не JSON или повреждён)
    2302 – Отсутствуют обязательные поля
    2303 – Несоответствие схеме (типы, значения)
    2304 – Частичный парсинг (не все поля извлечены)
24xx	Saver	Сохранение в jsonstore
    2400 – Данные успешно сохранены
    2401 – Ошибка записи (файловая система)
    2402 – Недостаточно прав
    2403 – Конфликт версий (данные изменены другим процессом)
    2404 – Частичное сохранение
25xx	Loader	Загрузка из jsonstore
    2500 – Данные успешно загружены
    2501 – Данные не найдены (по указанному ключу)
    2502 – Ошибка чтения хранилища
    2503 – Данные повреждены (битый JSON)
26xx	UpdateDecider	Проверка свежести данных
    2600 – Данные актуальны (обновление не требуется)
    2601 – Данные устарели (требуется обновление)
    2602 – Ошибка определения свежести (невозможно сравнить метки)
